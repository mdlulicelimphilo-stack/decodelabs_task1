## Description
A simple rule-based chatbot that responds to user inputs using a dictionary mapping. No machine learning, just pure logic.

## Features
- Responds to greetings (hello, hi, hey)
- Exit commands (bye, exit, quit)
- Input sanitization (lowercase, strip, punctuation removal)
- Hash map (dictionary) for O(1) lookup
- Fallback response for unknown inputs

## How to Run
```bash
python chatbot.py
